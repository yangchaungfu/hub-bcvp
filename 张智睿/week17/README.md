>第十七周作业-改动dl.py，识别用户重听需求并重播

>>1.使用 Jaccard 相似度函数：我添加了 jaccard_similarity() 函数，它通过计算两个字符串的 Jaccard 相似度来判断它们的相似度。

>>2.nlu() 方法改动：在 nlu() 方法中，我们使用了 Jaccard 相似度来模糊匹配用户输入和关键词库中的“重听”和“重看”相关表达。如果用户输入与关键词库中的某个表达式相似度超过设定的阈值（如 0.6），系统就识别为“重听”或“重看”请求，并返回上次的回复。

>>3.run() 方法改动：保持原有逻辑，在识别到“重听”或“重看”请求时返回最近一次的回复。
