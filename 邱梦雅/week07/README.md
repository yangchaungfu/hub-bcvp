### 3. 实验对比组
一共设置3组模型文本分类实验：
1. Group 1：对比非预训练模型
+ 选取模型：fast_text, cnn, gated_cnn, rnn, lst, gru, rcnn, stack_gated_cnn
2. Group 2：对比预训练模型
+ 选取模型：bert, bert_lstm, bert_cnn, bert_mid_layer
3. Group 3：对比非预训练模型和预训练模型
+ 选取模型：gated_cnn, lstm, rcnn, bert, bert_mid_layer
